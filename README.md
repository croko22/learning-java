# Learning Java

A comprehensive Java learning path from basics to advanced topics.

## 📚 Course Structure

### 01 - Java Basics
- Variables & Constants
- Data Types
- Operators
- Control Flow (if/else, switch)
- Loops (for, while, do-while)

### 02 - Object-Oriented Programming
- Classes & Objects
- Inheritance
- Polymorphism
- Encapsulation
- Interfaces
- Abstract Classes

### 03 - Collections Framework
- Lists (ArrayList, LinkedList)
- Sets (HashSet, TreeSet)
- Maps (HashMap, TreeMap)
- Queues

### 04 - Exception Handling
- Try-catch blocks
- Custom exceptions
- Best practices

### 05 - Generics
- Generic classes
- Generic methods
- Bounded types

### 06 - Lambdas & Streams
- Lambda expressions
- Stream API
- Functional interfaces

### 07 - Concurrency
- Threads
- Executors
- Synchronization

### 08 - I/O Operations
- File handling
- Reader/Writer
- Serialization

### 09 - Databases
- JDBC
- Connection pooling

### 10 - Web Development
- Servlets
- JSP basics

### 11 - Testing
- JUnit
- Test-driven development

## 🚀 Projects
- **Calculator** - Basic arithmetic operations
- **TodoList** - Task management app
- **StudentManagement** - CRUD operations

## 🛠️ Setup
```bash
# Clone the repository
git clone https://github.com/croko22/learning-java.git

# Navigate to a topic
cd 01-basics/variables

# Compile and run
javac Main.java
java Main
```

## 📖 How to Use
1. Start with `01-basics`
2. Move to `02-oop` for OOP concepts
3. Explore `03-collections` for data structures
4. Complete projects in `projects/`
